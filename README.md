# Core Flow — Telegram Bot & Mini App 💰📅

Умный ассистент в Telegram для контроля личных финансов и планирования задач с интегрированным веб-интерфейсом (Mini App). Проект реализован на чистом PHP с использованием архитектуры хэндлеров и базы данных SQLite.

## 🚀 Возможности

- **Управление финансами:** Быстрая запись расходов и доходов прямо в чат (например, `Кофе 300` или `+50000 Стипендия`).
- **Менеджер задач:** Добавление планов в текстовом формате (например, `Встреча завтра в 10`).
- **Telegram Mini App:** Интерактивный Dashboard в стиле macOS Midnight для просмотра детальной статистики и динамического управления задачами (выполнение тасок по клику на чекбокс без перезагрузки страницы через AJAX/Fetch API).

## 🛠️ Технологический стек

- **Backend:** PHP 8.x (ООП, PDO, Архитектура Handlers)
- **Database:** SQLite
- **Frontend:** HTML5, CSS3 (Grid/Flexbox, переменные темы Telegram), JavaScript (Fetch API, DOM-анимации)
- **Инструменты локального окружения:** Nginx, PHP-FPM (Homebrew macOS), ngrok (для туннелирования Webhook)
